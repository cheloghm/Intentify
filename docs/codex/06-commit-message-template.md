# Commit Message Template (Mandatory)

Title sentence.

- What changed (bullets)
- Why (bullets)
- Tests run (bullets)
- Security notes (bullets)

Example:
Added <component> scaffolding and tests.
- Implemented <x>...
- Added <y> tests...
- Tests: dotnet build -c Debug; dotnet test -c Debug
- Security: <note about secrets/guards>
